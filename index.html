Welcome to flippinLogoJS

This is a little snippet of JS and CSS to flip a logo or image on mouseover. It uses Jquery and CSS3 animations.

How to use flippinLogo

  • You only need the flippinLogo.js and flippinLogo.css files - everything else is generic
  • The flippinLogoJS is designed to flip an img element however it could be used on almost anything.
  • The img to be flipped should be given the class of .flippinLogo
  • You can add the class to various different elements on the page you want to flip.
  • Feel free to edit flippinLogo.css to change the animation.
